What idea does an open set capture?

A topology defines what subsets are open sets in a space. Assuming we have topological space (X, \scrT), the topology \scrT needs to satisfy three axioms below:

  • \varnothing is an open set, and X is an open set.
  • Assuming \set{U_i} is a family of arbitrary number of open sets, then \bigcup \set{U_i} is an open set.
  • Assuming \set{U_i} is a family of finite number of open sets, then \bigcap \set{U_i} is an open set.

These axioms feel quite enigmatic if not completely arbitrary. Some people draw an analogy between an open set and an open ball in metric space, where an open ball B_r(p) means a region close enough to a certain point p, with distance less than r. To see some problems in this interpretation, consider a disconnected open set like (0, 1) \cup (100, 101). We will find that it contains points that are far away from each other; More importantly, this interpretation doesn't distinguish between open sets and closed sets well, since closed sets like [0, 1] can also be considered as a region close to a point. Not to metion that this interpretion doesn't explain arbitrary union versus finite intersection at all. The goal of this article is to present a hopefully more satisfactory explanation on open sets.

The idea of boundary

When we are dealing with plain sets, we are given a clear-cut set membership relation - either in the set or not in the set. However, we will see that with the additional structure of a topology, things stop to behave in a binary way. The new relation allows some fuzziness, that is neither being strictly inside nor strictly outside. This is the idea of boundary.

To define boundary, we need to first define other concepts. A closed set is a complement of an open set, that is, if S \subseteq X is an open set, then X - S is a closed set. Because of this duality, we can also formulate topology by closed sets using De Morgan's laws. Thus a topology is a collection of closed sets that are closed under arbitrary intersection and finite union.

The interior of a set S is defined to be the union of all open sets contained in S, denoted as \inter{S}. The closure of a set is defined to be the intersection of all closed sets that contain S, denoted as \cl{S}. Because arbitrary union of open sets results in an open set, and arbitrary intersection of closed sets results in a closed set, what we actually defined is \inter{S} as the largest open set contained in S, and \cl{S} as the smallest closed set that contains S. That is to say, the \inter{S} and \cl{S} together form the tightest bound around set S. What is sandwiched between is defined to be the boundary, denoted as \bound{S} = \cl{S} - \inter{S}.

The dotted line illustrates the set S under consideration. It is bounded by its interior \inter{S} and closure \cl{S}, and the area between them is the boundary \bound{S}.

Points in boundary are hard to say if they belong to the set or not belong to the set, this is contrary to the set membership. Take a set S = [0, 1) where both 0 and 1 are the boundary points. Although by set membership we know for sure that 0 \in S and 1 \not\in S, however we cannot say anything assertive about membership of 0 and 1 in the topological sense.

Let's consider the property of redness in the space of colors. Scarlet is pretty red, crimson is also quite red. Ruby is less red, salmon and maroon are probably even less. Going a bit further we get colors like pink, orange and purple that might be the last colors that can still be seen as 'red'. Beyond those, colors like blue and green should never be considered as red. Thus we see this gradient of confidence in redness, from definitely being red, to still possibly being red (or not red), to never being red. We can easily see the analogy here. The interior contains colors that are definitely red, the closure contains color that are possibly red, therefore the boundary points are the colors in doubt.

An open set is a region that contains no boundary points, thus when we find a point in the set by set membership, we can be sure that the point is in the region, since it cannot be a boundary point. For example, being in the open set of redness means the color is definitely red. However, not being in the set doesn't necessarily means not in the region, since the complement is a closed set and it contains all the boundary points in doubt. In the color example, not definitely being red doesn't necessarily mean it is not red, it might be salmon or maroon that can still possibly be considered red.

Note the clopen sets (sets that are both open and closed) are the sets without boundary, hence no points in doubt. By the above analogy, a clopen set might correpond to some exact region without fuzziness like colors whose hue in HSV representation is within certain range. A different topology, however, might say this region still has uncertain boundary since what happens on the boundary of the range values?

In summary, open set gives us a language to talk about fuzzy membership like 'definitely in' or 'definitely not in'. A collection of open sets thus gives us a toolbox of positive assertions that we can know for sure about the topological membership in the space.

Reasons behind open set axioms

After knowing what idea an open set is trying to convey, let's have a look at why the axioms of open sets are reasonable. Recall that open sets represent regions such that if a point is in the set, then we know it is definitely in the region. When we are asked if a point is in the region, being in the open set can be seen as a proof for the positive answer, whereas a non-open set cannot serve as a proof, because the existence of boundary points doesn't allow us to state positive answer assertively.

Now consider the union of a collection of open sets \set{U_i}. We want to know if the union is an open set, the same question is can we know if a point is in the union region without doubt? We can do that because we can provide one known open set U_i to serve as the proof that the point is indeed in the region. It is fine that we don't know which U_i contains the test point, the important bit is that we can indeed find such an open set, and more importantly we only need one such open set.

For the case of intersection, things are a bit different. In order to know something is in the intersection region without doubt, we have to show the proof of being in individual region without doubt for every region. When there are infinite number of open sets in the collection, we would need infinitely long proofs. And this is prohibited in logical reasoning - we can reason about infinite things, but we cannot reason for infinitely long. This infinity effectively becomes a new type of boundary point that traps our positive assertion.

We can easily see why empty set and the entire space are both open sets. Empty set doesn't allow us to say anything assertively, thus it is an open set vacuously. Any point in the topological space definitely belongs to the space, since that is the entire universe under discussion, therefore the entire space is also an open set.

Hopefully these arguments make open set axioms a bit more reasonable. A lot of these ideas come from the first few chapters of the book 'Topology via Logic' by Steven Vickers, check it out if you are interested.
